HTML Cheatsheet

Essential HTML tags and examples for quick reference.

<html>

The root element that represents the entire HTML document.

<html></html>

<head>

Contains meta-information about the HTML document, such as the title, meta tags, and links to stylesheets or scripts.

<head></head>

<title>

Sets the title of the webpage, which appears in the browser tab.

<title>My Website</title>

<h1> to <h6>

Represents headings, with <h1> being the highest level and <h6> the lowest.

<h1>Main Heading</h1> <h2>Subheading</h2>

<p>

Defines a paragraph of text.

<p>This is a paragraph of text.</p>

<a>

Creates a hyperlink to another webpage or resource.

<a href="https://www.example.com">Visit Example</a>

<img>

Embeds an image in the webpage. Requires the src attribute to specify the image source.

<img src="image.jpg" alt="A description of the image">

<ul> and <li>

<ul> creates an unordered list, and <li> represents an individual list item.

<ul> <li>First item</li> <li>Second item</li> <li>Third item</li> </ul>

<div>

A container element that groups together sections of content for styling or scripting purposes.

<div></div>

<form>

Represents a form used to collect user input, which can then be submitted to a server for processing.

<form action="/submit" method="post"> <label for="name">Name:</label> <input type="text" id="name" name="name"> <input type="submit" value="Submit"> </form>
